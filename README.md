# ISU-HOTEL 易宿酒店

*--JPE小组--*

本仓库为携程大作业易宿酒店项目的实现源码，本项目整体采用前后端分离架构，包括了基于小程序移动端的信息查询预定系统、基于Web端的酒店商家管理系统以及服务器端的RESTFUL风格的统一接口平台。

## 整体文件结构

```
/backend  ---> 服务端(接口平台)源码
/miniapp  ---> 移动端(预定平台)源码
/web      ---> 网页端(管理平台)源码
README.md ---> 本文件
```

## 移动端

小程序移动端使用了基于Taro + React + TypeScript的技术栈构建了一套包含完整的用户预定流程与丰富的酒店信息查询功能的门面系统，通过对UI的整体设计与NutUI组件、地图组件的集成，本系统实现了用户交互流程的统一与流畅。另外，通过使用多种优化技术，本系统在工程方面也实现了良好的性能表现和代码质量。

## 网页端

网页端管理平台使用了React+Vite+TypeScript的技术栈，基于AntDesign提供的统一UI组件和React生态中Redux和React-Router的集成，实现了完整、高效的酒店商户信息管理与发布系统。

## 服务端

服务端使用基于Node.js的Nest.js+Prisma+TypeScript架构, 通过分层式的系统设计构建了一个良好的RESTFUL接口平台，在保证系统整体的灵活性同时也通过基于JWT的验证方法保证了用户数据权限的独立性和数据的安全性
